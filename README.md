# Iac-pulumi-gcp

## Getting started

- 1 Clone this into Gitlab repo
- 2 Update CICD Variabels
- 3 Update Pulumi.yaml with 
   - name: gcp-infra-dev >> - name: <your choice>
- 4 pulumi stack init <your choice>
